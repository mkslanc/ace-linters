Inline Autocompletion Commands

  • Start Autocomplete: Alt+C (Windows), Option+C (Mac)
  • Next Suggestion: Alt+]
  • Previous Suggestion: Alt+[
  • Accept Suggestion: Tab or Ctrl+Right (Windows), Tab or Cmd+Right (Mac)
  • Close Suggestions: Esc

GitHub Copilot Authentication

If GitHub Copilot code isn't copied automatically, please check your browser console (F12) for a message like:

{
    "userCode": "ABCD-EFGH",
    ...
}

Use the provided userCode to authenticate in the window that opens for GitHub Copilot sign-in.
